Java SDK for Teegon Cloud

构建安装
-----------------------------------------------
### 使用maven构建

- 下载安装websocket客户端依赖

```shell
git clone https://github.com/hashio/websocket-client.git
cd websocket-client
mvn clean install
```

- 下载安装teegon_java sdk

```shell
git clone https://github.com/daiyuok/teegon_java
cd teegon_java
mvn clean install
```
